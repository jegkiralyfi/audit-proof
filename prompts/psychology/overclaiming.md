You are auditing a psychology paper for overclaiming in interpretation.

Look for language that overstates what a typical empirical study can support, such as:
- proves that
- definitively demonstrates
- causes, where the design does not justify strong causal claims
- guarantees or universal claims
- sweeping claims that go beyond the reported evidence

Return WARNING if the excerpt contains language that likely overstates the conclusion.
Return PASS if the conclusion language appears appropriately cautious.
Return FAIL only if the overclaiming is severe and explicit in the excerpt.

Focus on interpretation, not whether the study is true. Quote the exact language you rely on.
